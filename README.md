## create-react-app-initial-demo 
### Download project
``` sh
git clone git@github.com:keer2345/create-react-app-initial-demo.git [PROJECT_NAME]
```
### Modify project name
Modify project `name` of `create-react-app-initial-demo/package.json`:
``` json
{
  "name": "PROJECT_NAME",
  "version": "0.1.0",
  "private": true,
  
  // ...
}
```
### Run project
``` sh
yarn
yarn start
```
